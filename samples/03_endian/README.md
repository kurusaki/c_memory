## この動画について

📺 **[動画はこちら](https://youtu.be/OTR_OkmE_l8)**

同じ「20」という値を代入しても、char、short、int、long ではメモリへの保存方法が異なります。
この動画では、前回作成した mdump 関数を使って、各型が実際にメモリ上でどのように保存されているのかを確認します。
また、なぜ「20」が「14」と表示されるのか、その理由も解説。最後には「なぜ 14 00 00 00 の順番で保存されるのか？」という次回につながる疑問も紹介します。
メモリの視点からC言語を理解したい方におすすめの内容です。
